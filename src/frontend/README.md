## 🛠️ Prerequisites

Before you start, make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18.0.0 or higher recommended)
- [npm](https://www.npmjs.com/) (comes pre-packaged with Node)

---

## ⚡ Setup & Installation

### 1. Install Dependencies
Navigate into the root of your project directory using your terminal and install the required project packages:

```bash
npm install

### 2. Run the Development Server
Start the local Vite development server:

```bash
npm run dev
